# Signal DB Summary

현재 프로젝트 구현 기준의 신호 요약입니다.
DBC 참고: `docs/Golf_6_PQ35.dbc`, `docs/VW_Passat_B6.dbc`
코드 기준: 각 보드의 `protocol_ids.h`, `engine_sim.c`, `bcm_signal.c`, `adas_signal.c`

| CAN ID | Message | 현재 의미 | 주요 필드 |
|---:|---|---|---|
| 0x100 | IGN Status | Board A IGN 기준 프레임 | byte[5] bit0 = IGN ON |
| 0x280 | Motor_1 / RPM | Board A RPM | byte[2..3] little-endian raw, rpm = raw / 4 |
| 0x1A0 | Speed keepalive | Board A가 현재 zero-filled keepalive로 송신 | 수신 측에는 legacy non-zero raw / 80 호환 코드가 남아 있음 |
| 0x5A0 | Speed reference | Board A speed 기준 | byte[2] = km/h / 2 |
| 0x288 | Motor_2 / Coolant | Board A coolant | byte[1], degC = raw * 0.75 - 48 |
| 0x481 | Engine Warning Status | Board A warning bitfield | byte[0] flags, byte[1] coolant, byte[2..3] rpm |
| 0x531 | Turn Status | Board D 방향지시등 | byte[2] bit0 left, bit1 right, bit2 hazard |
| 0x635 | Cluster Brightness | Board D 밝기 fade/hold | byte[0] 0..100 |
| 0x3A0 | ADAS_Status | Board E CAN1 내부 ADAS 상태 | flags, risk, distance, fault, speed, alive |
| 0x5D6 | Parking Assist | Board B가 ADAS 위험도 기준으로 CAN2에 생성 | risk >= 2 동안 byte[1] sweep |
| 0x390 | Legacy Body Status | Router 호환용 | 현재 Board D는 송신하지 않음 |
